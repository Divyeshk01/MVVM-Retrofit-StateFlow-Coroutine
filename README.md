# 🚀 MVVM + Retrofit Android Architecture

This project demonstrates a **modern Android architecture** using **MVVM (Model-View-ViewModel)** with **Retrofit** for API integration and **StateFlow** for reactive UI updates.

---

## 🏗️ Architecture Overview

The app follows a clean and scalable architecture:

UI (Activity)
        ↓
ViewModel (StateFlow)
        ↓
Repository
        ↓
Retrofit API Service
        ↓
Remote Server

---

## 🧩 Tech Stack

- 🟢 Kotlin
- 🟢 MVVM Architecture
- 🟢 Retrofit (REST API calls)
- 🟢 Coroutines & StateFlow
- 🟢 ViewModel (Android Jetpack)
- 🟢 Gson / Moshi (JSON parsing)
